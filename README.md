# Personal Portfolio — Siyamukelo Cebekulu

A responsive, multi-page portfolio website showcasing backend development experience, project highlights, and contact form validation.

## Key Features

- Responsive navigation bar with mobile menu
- Hero section and profile summary
- Projects, Work Experience, Education, and Testimonials
- Contact page with JavaScript form validation
- Card-based layout with clean visual hierarchy
- Google Fonts integration
- CSS variables for easy theming
- Dark mode toggle with saved preference
- Ready for Three.js background and typing animation extensions
- Multi-page structure: `index.html`, `about.html`, `projects.html`, `contact.html`
